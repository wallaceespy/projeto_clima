# 🌤️ Projeto Previsão do Tempo

Aplicação web desenvolvida com **HTML, CSS e JavaScript** que consome a API Open-Meteo para exibir informações climáticas em tempo real e previsão para os próximos dias.

---

## 🚀 Funcionalidades

* 🔍 Busca de cidade
* 🌡️ Temperatura atual
* 🌧️ Animação de chuva em tempo real
* ⚡ Relâmpagos em tempestades
* 📅 Previsão para 5 dias
* 🎨 Interface moderna e responsiva

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla)
* API Open-Meteo

---

## 📦 Como rodar o projeto

1. Clone o repositório:

git clone https://github.com/seu-usuario/projeto_clima.git

2. Acesse a pasta:

cd projeto_clima

3. Abra o arquivo `index.html` no navegador

---

## 🌍 Como usar

1. Digite o nome de uma cidade
2. Clique em **Buscar**
3. Veja:

   * Temperatura atual
   * Condição climática
   * Previsão dos próximos dias

---

## 🔐 Segurança e Privacidade

* Nenhum dado do usuário é armazenado
* A aplicação consome apenas APIs públicas
* Não utiliza autenticação ou dados sensíveis
* A localização não é coletada automaticamente

---

## ⚠️ Aviso de Privacidade

Este aplicativo utiliza a API Open-Meteo para buscar dados climáticos.
Nenhuma informação pessoal é coletada, armazenada ou compartilhada.
