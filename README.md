# 🌤️ Projeto Previsão do Tempo

Aplicação web desenvolvida com **HTML, CSS e JavaScript** que consome a API Open-Meteo para exibir informações climáticas em tempo real e previsão para os próximos dias.

---

## 🚀 Funcionalidades

* 🔍 Busca de cidade
* 🌡️ Temperatura atual
* 🌧️ Animação de chuva em tempo real
* ⚡ Relâmpagos em tempestades
* 📅 Previsão para 5 dias
* 🎨 Interface moderna e responsiva

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla)
* API Open-Meteo

---

## 📦 Como rodar o projeto

1. Clone o repositório:

git clone https://github.com/seu-usuario/projeto_clima.git

2. Acesse a pasta:

cd projeto_clima

3. Abra o arquivo `index.html` no navegador

---

## 🌍 Como usar

1. Digite o nome de uma cidade
2. Clique em **Buscar**
3. Veja:

   * Temperatura atual
   * Condição climática
   * Previsão dos próximos dias

---

## 🔐 Segurança e Privacidade

* Nenhum dado do usuário é armazenado
* A aplicação consome apenas APIs públicas
* Não utiliza autenticação ou dados sensíveis
* A localização não é coletada automaticamente

---

## ⚠️ Aviso de Privacidade

Este aplicativo utiliza a API Open-Meteo para buscar dados climáticos.
Nenhuma informação pessoal é coletada, armazenada ou compartilhada.

# 🔐 Auditoria de Segurança e Privacidade

## ✔️ Análise

O projeto foi avaliado com base em boas práticas de segurança para aplicações web.

---

## 🔍 Riscos Identificados

1. Entrada do usuário sem validação adequada
2. Possível erro caso a API não retorne dados
3. Uso de innerHTML (risco leve de XSS)

---

## ✅ Correções Aplicadas

* Validação de campo vazio
* Uso de try/catch
* Tratamento de erros da API
* Limpeza de conteúdo antes de renderizar

---

## 🔒 Recomendações

* Utilizar HTTPS em produção
* Implementar Content Security Policy (CSP)
* Evitar uso excessivo de innerHTML
* Validar todos os inputs do usuário

---

## 🛡️ Ambiente de Produção

* Hospedar em servidor seguro (HTTPS)
* Monitorar erros
* Manter dependências atualizadas

---

## 📌 Conclusão

O projeto é seguro para uso educacional e demonstra boas práticas iniciais.
